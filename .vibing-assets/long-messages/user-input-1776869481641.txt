# Flow Manager MCP App for Claude Code

## Overview
This MCP app lets Claude Code **trigger workflow runs** and **check status** against your flow manager backend. Users can say "Run my content pipeline with topic X" or "What's the status of run abc123".

## Backend API Requirements
Your backend needs these endpoints:

```
POST /api/runs
  Body: { "workflowId": "flow_abc123", "input": { "topic": "AI agents", "audience": "developers" } }
  Returns: { "runId": "run_xyz789", "status": "pending" }

GET  /api/runs/:runId
  Returns: { "status": "running", "progress": {...}, "logs": [...] }
```

## MCP Server Implementation

### 1. Node.js MCP Server (`flow-manager-mcp.js`)

```javascript
#!/usr/bin/env node
import { createServer } from '@modelcontextprotocol/sdk/server/index.js';
import { z } from 'zod';

const server = await createServer({
  name: 'flow-manager',
  version: '1.0.0'
});

// Tool 1: Start a workflow run
server.tool('start_workflow', {
  description: 'Start a new workflow run with input parameters',
  inputSchema: z.object({
    workflowId: z.string().describe('Workflow ID from the flow manager'),
    input: z.record(z.any()).describe('Input data for the workflow')
  })
}, async ({ workflowId, input }) => {
  const response = await fetch('https://your-flow-manager.com/api/runs', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ workflowId, input })
  });
  
  const result = await response.json();
  return {
    runId: result.runId,
    status: result.status,
    message: `Started workflow ${workflowId}. Run ID: ${result.runId}`
  };
});

// Tool 2: Get run status
server.tool('get_run_status', {
  description: 'Check the status of a workflow run',
  inputSchema: z.object({
    runId: z.string().describe('Run ID returned from start_workflow')
  })
}, async ({ runId }) => {
  const response = await fetch(`https://your-flow-manager.com/api/runs/${runId}`);
  const result = await response.json();
  
  return {
    runId,
    status: result.status,
    progress: result.progress,
    logs: result.logs?.slice(-5) || [], // Last 5 log entries
    completed: result.completed_at ? true : false
  };
});

// Tool 3: List recent runs (bonus)
server.tool('list_recent_runs', {
  description: 'List the 5 most recent workflow runs',
  inputSchema: z.object({})
}, async () => {
  // Optional: implement /api/runs/recent endpoint
  return { runs: [], message: 'Recent runs endpoint not implemented' };
});

await server.connect();
```

## Installation & Setup

### Install MCP Server
```bash
npm init -y
npm install @modelcontextprotocol/sdk zod
# Save the server code as flow-manager-mcp.js
```

### Connect to Claude Code
```bash
# Add as local MCP server
claude mcp add flow-manager --type stdio --command "node" --args "flow-manager-mcp.js"

# Or use npx for quick testing
claude mcp add flow-manager --type stdio --command "npx" --args "-y" "your-mcp-package"
```

### Environment Variables
```bash
export FLOW_MANAGER_URL="https://your-flow-manager.com"
export FLOW_MANAGER_API_KEY="your-api-key"
```

## Backend API Implementation

Your flow manager needs these Express.js routes:

```javascript
// POST /api/runs
app.post('/api/runs', async (req, res) => {
  const { workflowId, input } = req.body;
  const runId = generateRunId();
  
  // Queue the workflow run
  await startWorkflow(runId, workflowId, input);
  
  res.json({ runId, status: 'pending' });
});

// GET /api/runs/:runId
app.get('/api/runs/:runId', async (req, res) => {
  const run = await getRunStatus(req.params.runId);
  res.json(run);
});
```

## Error Handling

The MCP server automatically handles timeouts and retries. Add validation:

```javascript
server.tool('start_workflow', {
  // ...
}, async ({ workflowId, input }) => {
  if (!workflowId) {
    throw new Error('workflowId is required');
  }
  
  try {
    const response = await fetch(...);
    if (!response.ok) throw new Error(`HTTP ${response.status}`);
    return await response.json();
  } catch (error) {
    return {
      error: error.message,
      runId: null
    };
  }
});
```
